# Nextstrain build for ANDV

## About 

This repository provides scripts for the phylogenetic analysis of the ANDV virus. ANDV is a multi-segmented RNA-virus with three segments (L, M and S). Phylogenetic analysis is performed on each segment using the references; [NC_003468.2](https://www.ncbi.nlm.nih.gov/nuccore/NC_003468.2) for segment L, [NC_003467.2](https://www.ncbi.nlm.nih.gov/nuccore/NC_003467.2) for segment M and [NC_003466.1](https://www.ncbi.nlm.nih.gov/nuccore/NC_003466.1) for segment S. This repository creates Nextstrain builds without linking the segments.

This repository and these builds are under active development and will be improved in the coming months and days. Please report any suspected problems as issues via Github or directly to Emma.

## Data

We source sequence data and metadata from [Pathoplexus](https://pathoplexus.org) which ingests data from INSDC and provides data from INSDC together with data that were submitted directly to Pathoplexus. 

Some data in this build may be RESTRICTED-USE - meaning its use in preprints or publications is restricted. Please see the Pathoplexus [Data Use Terms](https://pathoplexus.org/about/terms-of-use/data-use-terms) for more details. 

The data underlying this build can be accessed directly at [Pathoplexus](https://pathoplexus.org/andv/).

We gratefully acknowledge the authors, originating and submitting laboratories of the genetic sequences and metadata for sharing their work.
